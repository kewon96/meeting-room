# 공통 용어집 (Glossary)

회의실 전체에서 공유하는 용어·약어. 주제별로만 쓰이는 용어는 해당 주제의 `CLAUDE.md` 에 정의.

| 용어 | 정의 | 비고 |
|------|------|------|
| 회의실 | 본 디렉토리 (`meeting-room/`) | |
| 주제 | `topics/<slug>/` 한 폴더 단위 | 1주제 = 1폴더 |
| 페르소나 | `.claude/agents/` 의 서브에이전트 정의 | facilitator/scribe/... |
| 회의록 (minutes) | 결정·action item 만 담은 정제된 기록 | |
| 토론 로그 (discussions) | 자유 발언·탐색의 원본 기록 | |
| 산출물 (artifacts) | 회의 결과로 만들어진 문서·코드·스펙 | |
